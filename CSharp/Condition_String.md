# 문자열에 따른 두 수 비교하기

## 문제 설명

두 문자열 `ineq`와 `eq`, 그리고 두 정수 `n`, `m`이 주어집니다.

`ineq`는 부등호 방향을 나타내며, `eq`는 등호 포함 여부를 나타냅니다.

| `ineq` | `eq`  | 비교 조건    |
| ------ | ----- | -------- |
| `">"`  | `"="` | `n >= m` |
| `"<"`  | `"="` | `n <= m` |
| `">"`  | `"!"` | `n > m`  |
| `"<"`  | `"!"` | `n < m`  |

조건이 참이면 `1`, 거짓이면 `0`을 반환하는 `solution` 함수를 작성합니다.

---

## 제한 사항

* `1 ≤ n, m ≤ 100`
* `ineq`는 `"<"` 또는 `">"` 중 하나입니다.
* `eq`는 `"="` 또는 `"!"` 중 하나입니다.

---

## 입출력 예

| ineq  | eq    |  n |  m | result |
| ----- | ----- | -: | -: | -----: |
| `"<"` | `"="` | 20 | 50 |      1 |
| `">"` | `"!"` | 41 | 78 |      0 |

---

## 입출력 예 설명

### 예제 1

```text
ineq = "<"
eq = "="
n = 20
m = 50
```

`ineq`가 `"<"`이고 `eq`가 `"="`이므로 비교 조건은 다음과 같습니다.

```text
20 <= 50
```

조건이 참이므로 `1`을 반환합니다.

### 예제 2

```text
ineq = ">"
eq = "!"
n = 41
m = 78
```

`ineq`가 `">"`이고 `eq`가 `"!"`이므로 비교 조건은 다음과 같습니다.

```text
41 > 78
```

조건이 거짓이므로 `0`을 반환합니다.

---

## 해결 방법

`ineq`와 `eq`의 조합에 따라 비교 연산을 다르게 수행하면 됩니다.

가능한 조합은 총 4가지입니다.

1. `ineq == ">"` 그리고 `eq == "="`
   → `n >= m`

2. `ineq == "<"` 그리고 `eq == "="`
   → `n <= m`

3. `ineq == ">"` 그리고 `eq == "!"`
   → `n > m`

4. `ineq == "<"` 그리고 `eq == "!"`
   → `n < m`

각 조건을 `if`문으로 확인한 뒤, 참이면 `1`, 거짓이면 `0`을 반환합니다.

---

## C# 코드

```csharp
using System;

public class Solution
{
    public int solution(string ineq, string eq, int n, int m)
    {
        // ">"와 "=" 조합이면 n >= m 비교
        if (ineq == ">" && eq == "=")
        {
            return n >= m ? 1 : 0;
        }

        // "<"와 "=" 조합이면 n <= m 비교
        if (ineq == "<" && eq == "=")
        {
            return n <= m ? 1 : 0;
        }

        // ">"와 "!" 조합이면 n > m 비교
        if (ineq == ">" && eq == "!")
        {
            return n > m ? 1 : 0;
        }

        // "<"와 "!" 조합이면 n < m 비교
        if (ineq == "<" && eq == "!")
        {
            return n < m ? 1 : 0;
        }

        // 문제 조건상 이 부분은 실행되지 않지만,
        // 모든 경로에서 값을 반환하기 위해 작성
        return 0;
    }
}
```

## 더 간단한 코드

조건식을 하나의 문자열로 합친 뒤 비교할 수도 있습니다.

```csharp
using System;

public class Solution
{
    public int solution(string ineq, string eq, int n, int m)
    {
        string condition = ineq + eq;

        if (condition == ">=")
        {
            return n >= m ? 1 : 0;
        }

        if (condition == "<=")
        {
            return n <= m ? 1 : 0;
        }

        if (condition == ">!")
        {
            return n > m ? 1 : 0;
        }

        if (condition == "<!")
        {
            return n < m ? 1 : 0;
        }

        return 0;
    }
}
```

| -------- | ------------------------------ |

## 삼항 연산자 설명

다음 코드는 삼항 연산자를 사용한 코드입니다.

```csharp
return n >= m ? 1 : 0;
```

위 코드는 아래 코드와 같은 의미입니다.

```csharp
if (n >= m)
{
    return 1;
}
else
{
    return 0;
}
```

즉, 조건이 참이면 `1`, 거짓이면 `0`을 반환합니다.

## 참고 사항

이 문제는 복잡한 알고리즘보다는 조건문을 정확히 작성하는 것이 핵심입니다.

특히 `eq`가 `"="`일 때는 등호를 포함한 비교를 해야 하고, `eq`가 `"!"`일 때는 등호를 포함하지 않는 비교를 해야 합니다.

따라서 다음 차이를 정확히 구분해야 합니다.

| 조건   | 의미     |
| ---- | ------ |
| `>=` | 크거나 같다 |
| `<=` | 작거나 같다 |
| `>`  | 크다     |
| `<`  | 작다     |
