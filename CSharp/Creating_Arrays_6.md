# 문자열 정수 변환하기

## 문제 설명

문자열 배열 `intStrs`와 정수 `k`, `s`, `l`이 주어집니다.

`intStrs`의 각 문자열에서 `s`번 인덱스부터 시작하는 길이 `l`짜리 부분 문자열을 잘라낸 뒤, 이를 정수로 변환합니다.

변환한 정수값이 `k`보다 큰 경우에만 결과 배열에 담아 반환하는 문제입니다.

---

## 입출력 예시

| intStrs | k | s | l | result |
|---|---:|---:|---:|---|
| `["0123456789", "9876543210", "9999999999999"]` | `50000` | `5` | `5` | `[56789, 99999]` |

---

## 문제 풀이 아이디어

각 문자열에 대해 다음 과정을 반복합니다.

1. 문자열에서 `s`번 인덱스부터 길이 `l`만큼 자릅니다.
2. 자른 문자열을 정수로 변환합니다.
3. 변환한 정수가 `k`보다 크면 결과 배열에 추가합니다.

예를 들어,

```text
"0123456789"에서 s = 5, l = 5
```

이면 잘라낸 문자열은 `"56789"`입니다.

이를 정수로 변환하면 `56789`이고, `56789 > 50000`이므로 결과 배열에 추가합니다.

---

## C# 풀이

```csharp
using System;
using System.Collections.Generic;

public class Solution
{
    public int[] solution(string[] intStrs, int k, int s, int l)
    {
        List<int> answer = new List<int>();

        foreach (string str in intStrs)
        {
            // s번 인덱스부터 길이 l만큼 문자열 자르기
            string subStr = str.Substring(s, l);

            // 자른 문자열을 정수로 변환
            int number = int.Parse(subStr);

            // k보다 큰 값만 결과 리스트에 추가
            if (number > k)
            {
                answer.Add(number);
            }
        }

        return answer.ToArray();
    }
}
```

---

## Python 풀이

```python
def solution(intStrs, k, s, l):
    answer = []

    for string in intStrs:
        # s번 인덱스부터 s + l 전까지 문자열 자르기
        sub_str = string[s:s + l]

        # 자른 문자열을 정수로 변환
        number = int(sub_str)

        # k보다 큰 값만 결과 리스트에 추가
        if number > k:
            answer.append(number)

    return answer
```

## 코드 동작 과정

입력값이 다음과 같다고 가정합니다.

```text
intStrs = ["0123456789", "9876543210", "9999999999999"]
k = 50000
s = 5
l = 5
```

각 문자열을 처리하면 다음과 같습니다.

| idx | 문자열 | 잘라낸 문자열 | 정수 변환값 | k보다 큰가? | 결과 |
|---:|---|---|---:|---|---|
| 0 | `"0123456789"` | `"56789"` | `56789` | Yes | `[56789]` |
| 1 | `"9876543210"` | `"43210"` | `43210` | No | `[56789]` |
| 2 | `"9999999999999"` | `"99999"` | `99999` | Yes | `[56789, 99999]` |

따라서 최종 결과는 다음과 같습니다.

```text
[56789, 99999]
```

## 사용한 주요 문법

### C#

| 문법 | 설명 |
|---|---|
| `Substring(start, length)` | 문자열에서 `start` 인덱스부터 `length` 길이만큼 자릅니다. |
| `int.Parse()` | 문자열을 정수로 변환합니다. |
| `List<int>` | 크기가 동적으로 변하는 정수 리스트입니다. |
| `ToArray()` | 리스트를 배열로 변환합니다. |
| `foreach` | 배열이나 리스트의 원소를 하나씩 순회합니다. |

### Python

| 문법 | 설명 |
|---|---|
| `string[s:s+l]` | 문자열에서 `s`부터 `s + l` 전까지 자릅니다. |
| `int()` | 문자열을 정수로 변환합니다. |
| `append()` | 리스트에 값을 추가합니다. |
| `for` | 리스트의 원소를 하나씩 순회합니다. |

## 시간 복잡도

`intStrs`의 길이를 `n`이라고 할 때, 각 문자열마다 길이 `l`만큼 자르고 정수로 변환합니다.

제한사항에서 `l <= 8`이므로 사실상 상수 시간으로 볼 수 있습니다.

```text
시간 복잡도: O(n)
공간 복잡도: O(n)
```

결과 배열에 최대 `n`개의 값이 들어갈 수 있으므로 공간 복잡도는 `O(n)`입니다.

---

## 정리

이 문제는 문자열 자르기와 정수 변환을 활용하는 기본 구현 문제입니다.

핵심은 다음과 같습니다.

```text
부분 문자열 추출 → 정수 변환 → k보다 큰 값만 저장
```

C#에서는 `Substring()`, Python에서는 슬라이싱을 사용하면 간단하게 해결할 수 있습니다.
