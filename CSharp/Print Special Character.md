# 특수문자 출력하기 (Print Special Characters)

---

## 문제 설명

다음과 같이 특수문자를 그대로 출력하는 C# 코드를 작성합니다.

---

## 출력 예시

```text
!@#$%^&*(\'"<>?:;
```

---

## 프로그램 설명

이 문제는 C#에서 **문자열(string)** 안에 특수문자를 포함하여 출력하는 방법을 연습하는 문제입니다.

특히 출력해야 하는 문자열에는 다음 문자가 포함되어 있습니다.

- 작은따옴표 `'`
- 큰따옴표 `"`
- 역슬래시 `\`

C#에서 큰따옴표 `"`와 역슬래시 `\`는 문자열 안에서 특별한 의미를 가지므로, 그대로 출력하려면 **이스케이프 문자(Escape Character)** 를 사용해야 합니다.

---

## 소스 코드

```csharp
using System;

class Program
{
    static void Main()
    {
        // 특수문자들을 그대로 출력합니다.
        // \" 는 큰따옴표(")를 출력하기 위한 표현입니다.
        // \\ 는 역슬래시(\)를 출력하기 위한 표현입니다.
        Console.WriteLine("!@#$%^&*(\\'\"<>?:;");
    }
}
```

---

## 주요 메서드 및 문법 설명

| 문법 / 메서드 | 설명 |
|---|---|
| `using System;` | `Console` 클래스를 사용하기 위해 필요한 네임스페이스입니다. |
| `class Program` | C# 프로그램의 기본 클래스입니다. |
| `static void Main()` | 프로그램이 시작되는 진입점(Entry Point)입니다. |
| `Console.WriteLine()` | 괄호 안의 내용을 출력한 뒤 줄바꿈합니다. |
| `\\` | 역슬래시 `\`를 출력하기 위한 이스케이프 표현입니다. |
| `\"` | 큰따옴표 `"`를 출력하기 위한 이스케이프 표현입니다. |

---

## 입력 예시

입력은 없습니다.

```text

```

---

## 출력 예시

```text
!@#$%^&*(\'"<>?:;
```

---

## 코드 동작 과정

1. 프로그램이 실행되면 `Main()` 메서드가 호출됩니다.
2. `Console.WriteLine()`이 실행됩니다.
3. 문자열 안에 작성된 특수문자들이 출력됩니다.
4. `\\`는 실제 출력에서 `\`로 출력됩니다.
5. `\"`는 실제 출력에서 `"`로 출력됩니다.

---

## 참고 사항

C# 문자열은 큰따옴표 `"`로 감싸서 작성합니다.

따라서 문자열 내부에 큰따옴표를 출력하려면 다음과 같이 작성해야 합니다.

```csharp
Console.WriteLine("\"");
```

역슬래시를 출력하려면 다음과 같이 작성해야 합니다.

```csharp
Console.WriteLine("\\");
```

이 문제에서는 역슬래시와 큰따옴표가 모두 포함되어 있으므로 다음과 같이 작성합니다.

```csharp
Console.WriteLine("!@#$%^&*(\\'\"<>?:;");
```
