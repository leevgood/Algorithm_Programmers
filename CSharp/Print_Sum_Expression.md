# 두 수의 합 출력하기 (Print Sum Expression)

---

## 📌 프로그램 설명

두 정수 `a`, `b`가 주어졌을 때, 두 수의 합을 계산한 뒤 아래와 같은 형식으로 출력하는 C# 프로그램입니다.

```text
a + b = c
```

예를 들어 `4 5`가 입력되면 다음과 같이 출력됩니다.

```text
4 + 5 = 9
```

---

## ✅ 제한사항

- `1 ≤ a, b ≤ 100`
- 입력은 공백으로 구분된 두 정수입니다.
- 출력 형식은 문제에서 제시한 형태와 정확히 일치해야 합니다.

---

## 💻 소스 코드

```csharp
using System;

public class Example
{
    public static void Main()
    {
        // 공백으로 구분된 두 정수를 입력받습니다.
        string[] input = Console.ReadLine().Split();

        // 입력받은 문자열을 정수로 변환합니다.
        int a = int.Parse(input[0]);
        int b = int.Parse(input[1]);

        // 두 정수의 합을 계산합니다.
        int c = a + b;

        // 문제에서 요구한 형식으로 출력합니다.
        Console.WriteLine($"{a} + {b} = {c}");
    }
}
```

---

## 🔍 주요 메서드 및 문법 설명

| 문법 / 메서드 | 설명 |
|---|---|
| `Console.ReadLine()` | 콘솔에서 한 줄의 입력을 받습니다. |
| `Split()` | 문자열을 공백 기준으로 나누어 문자열 배열로 만듭니다. |
| `int.Parse()` | 문자열을 정수형 `int`로 변환합니다. |
| `a + b` | 두 정수의 합을 계산합니다. |
| `$"{변수}"` | 문자열 보간(String Interpolation) 문법으로, 문자열 안에 변수 값을 쉽게 넣을 수 있습니다. |
| `Console.WriteLine()` | 결과를 콘솔에 출력합니다. |

---

## 📥 입력 예시

```text
4 5
```

---

## 📤 출력 예시

```text
4 + 5 = 9
```

---

## 🧠 코드 동작 과정

1. `Console.ReadLine()`으로 한 줄의 입력을 받습니다.
2. `Split()`을 사용해 입력값을 공백 기준으로 나눕니다.
3. `int.Parse()`를 사용해 문자열을 정수로 변환합니다.
4. `a + b`를 계산하여 변수 `c`에 저장합니다.
5. 문자열 보간을 사용해 `a + b = c` 형식으로 출력합니다.

---

## 📎 참고 사항

- 출력 형식에서 공백도 중요합니다.
- `+`와 `=` 앞뒤에 공백을 포함해야 예시와 동일한 출력이 됩니다.
- 문자열 보간을 사용하면 여러 변수와 문자를 깔끔하게 출력할 수 있습니다.
