# 주사위 게임 2 - C# 풀이

## 문제 설명

1부터 6까지의 숫자가 적힌 주사위 3개를 굴렸을 때 나온 숫자를 각각 `a`, `b`, `c`라고 합니다.

세 숫자의 관계에 따라 다음과 같이 점수를 계산합니다.

| 조건 | 점수 계산 |
|---|---|
| 세 숫자가 모두 다름 | `a + b + c` |
| 두 숫자만 같음 | `(a + b + c) × (a² + b² + c²)` |
| 세 숫자가 모두 같음 | `(a + b + c) × (a² + b² + c²) × (a³ + b³ + c³)` |

---

## 제한사항

- `a`, `b`, `c`는 1 이상 6 이하의 정수입니다.

---

## 입출력 예시

| a | b | c | result |
|---|---|---|---|
| 2 | 6 | 1 | 9 |
| 5 | 3 | 3 | 473 |
| 4 | 4 | 4 | 110592 |

---

## 소스 코드

```csharp
using System;

public class Solution
{
    public int solution(int a, int b, int c)
    {
        // 1제곱 합
        int sum1 = a + b + c;

        // 2제곱 합
        int sum2 = a * a + b * b + c * c;

        // 3제곱 합
        int sum3 = a * a * a + b * b * b + c * c * c;

        // 세 숫자가 모두 같은 경우
        if (a == b && b == c)
        {
            return sum1 * sum2 * sum3;
        }

        // 세 숫자가 모두 다른 경우
        if (a != b && b != c && a != c)
        {
            return sum1;
        }

        // 두 숫자만 같은 경우
        return sum1 * sum2;
    }
}
```

---

## 코드 설명

### 1. 각 제곱 합 계산

```csharp
int sum1 = a + b + c;
int sum2 = a * a + b * b + c * c;
int sum3 = a * a * a + b * b * b + c * c * c;
```

각 조건에서 반복적으로 사용되는 값을 미리 계산합니다.

| 변수 | 의미 |
|---|---|
| `sum1` | `a + b + c` |
| `sum2` | `a² + b² + c²` |
| `sum3` | `a³ + b³ + c³` |

---

### 2. 세 숫자가 모두 같은 경우

```csharp
if (a == b && b == c)
{
    return sum1 * sum2 * sum3;
}
```

`a`, `b`, `c`가 모두 같다면 문제 조건에 따라 다음 값을 반환합니다.

```csharp
(a + b + c) * (a² + b² + c²) * (a³ + b³ + c³)
```

예를 들어 `a = 4`, `b = 4`, `c = 4`라면:

```text
sum1 = 4 + 4 + 4 = 12
sum2 = 16 + 16 + 16 = 48
sum3 = 64 + 64 + 64 = 192

result = 12 × 48 × 192 = 110592
```

---

### 3. 세 숫자가 모두 다른 경우

```csharp
if (a != b && b != c && a != c)
{
    return sum1;
}
```

세 숫자가 모두 다르다면 `a + b + c`만 반환합니다.

예를 들어 `a = 2`, `b = 6`, `c = 1`이라면:

```text
2 + 6 + 1 = 9
```

---

### 4. 두 숫자만 같은 경우

```csharp
return sum1 * sum2;
```

위의 두 조건에 해당하지 않는다면, 남은 경우는 두 숫자만 같은 경우입니다.

따라서 다음 값을 반환합니다.

```csharp
(a + b + c) * (a² + b² + c²)
```

예를 들어 `a = 5`, `b = 3`, `c = 3`이라면:

```text
sum1 = 5 + 3 + 3 = 11
sum2 = 25 + 9 + 9 = 43

result = 11 × 43 = 473
```

## 코드 동작 과정

입력값이 다음과 같다고 가정합니다.

```csharp
a = 5;
b = 3;
c = 3;
```

1. `sum1 = 5 + 3 + 3 = 11`
2. `sum2 = 25 + 9 + 9 = 43`
3. `sum3 = 125 + 27 + 27 = 179`
4. 세 숫자가 모두 같지 않으므로 첫 번째 조건은 거짓
5. 세 숫자가 모두 다르지도 않으므로 두 번째 조건도 거짓
6. 두 숫자만 같은 경우이므로 `sum1 * sum2` 반환
7. 결과는 `11 * 43 = 473`

## 다른 풀이 방법

숫자의 종류 개수를 이용해서도 풀 수 있습니다.

```csharp
using System;
using System.Linq;

public class Solution
{
    public int solution(int a, int b, int c)
    {
        int count = new int[] { a, b, c }.Distinct().Count();

        int sum1 = a + b + c;
        int sum2 = a * a + b * b + c * c;
        int sum3 = a * a * a + b * b * b + c * c * c;

        if (count == 1)
        {
            return sum1 * sum2 * sum3;
        }

        if (count == 2)
        {
            return sum1 * sum2;
        }

        return sum1;
    }
}
```

## 풀이 핵심

이 문제는 주사위 숫자 3개의 중복 여부를 확인하는 문제입니다.

핵심은 다음 세 가지 경우를 정확하게 구분하는 것입니다.

1. 세 숫자가 모두 같은 경우
2. 두 숫자만 같은 경우
3. 세 숫자가 모두 다른 경우

조건문을 이용해 경우를 나눈 뒤, 문제에서 요구하는 계산식을 그대로 적용하면 됩니다.
