# 문자열로 주어진 수를 9로 나눈 나머지 구하기

## 문제 설명

음이 아닌 정수가 문자열 `number`로 주어집니다.

정수를 직접 숫자형으로 변환하지 않고, 다음 성질을 이용하여 `number`를 9로 나눈 나머지를 구해야 합니다.

> 어떤 수를 9로 나눈 나머지는 그 수의 각 자리 숫자의 합을 9로 나눈 나머지와 같습니다.

예를 들어 `123`의 각 자리 숫자의 합은 다음과 같습니다.

```text
1 + 2 + 3 = 6
```

따라서 `123 % 9 = 6`입니다.

---

## 제한사항

| 조건           | 내용                            |
| ------------ | ----------------------------- |
| `number`의 길이 | `1 ≤ number.Length ≤ 100,000` |
| 문자 구성        | 숫자로만 이루어짐                     |
| 시작 문자        | 정수 `0`이 아니라면 `'0'`으로 시작하지 않음  |

---

## 입출력 예시

| number                   | result |
| ------------------------ | -----: |
| `"123"`                  |    `6` |
| `"78720646226947352489"` |    `2` |

---

## 문제 풀이 아이디어

`number`의 길이는 최대 `100,000`입니다.

따라서 문자열 전체를 `int`, `long` 같은 숫자형으로 변환하면 범위를 초과할 수 있습니다.

예를 들어 C#의 `int`는 최대 약 21억까지만 저장할 수 있습니다.

```text
2,147,483,647
```

하지만 문제에서 주어지는 문자열은 훨씬 더 길 수 있으므로, 숫자 전체를 변환하면 안 됩니다.

대신 문자열을 한 글자씩 확인하면서 각 자리 숫자를 더합니다.

그 후 합계를 `9`로 나눈 나머지를 반환합니다.

---

## C# 풀이

```csharp
using System;

public class Solution {
    public int solution(string number) {
        int result = 0;

        foreach (char c in number) {
            // 문자 숫자를 실제 정수 숫자로 변환
            result += c - '0';
        }

        return result % 9;
    }
}
```

---

## C# 코드 설명

| 코드                           | 설명                             |
| ---------------------------- | ------------------------------ |
| `foreach (char c in number)` | 문자열 `number`의 각 문자를 하나씩 순회합니다. |
| `c - '0'`                    | 문자 숫자를 정수 숫자로 변환합니다.           |
| `result += c - '0'`          | 각 자리 숫자를 누적해서 더합니다.            |
| `return result % 9`          | 자리수 합을 9로 나눈 나머지를 반환합니다.       |

---

## C#에서 `c - '0'`을 사용하는 이유

문자 `'7'`은 숫자 `7`이 아니라 문자입니다.

하지만 문자 `'0'`부터 `'9'`까지는 연속된 문자 코드값을 가지고 있습니다.

따라서 다음과 같은 계산이 가능합니다.

```text
'7' - '0' = 7
```

즉, 문자 숫자를 실제 정수값으로 바꿀 수 있습니다.

---

## Python 풀이

```python
def solution(number):
    answer = 0
    
    for c in number:
        # 문자 숫자를 정수로 변환해서 더함
        answer += int(c)
    
    return answer % 9
```

---

## Python 코드 설명

| 코드                  | 설명                             |
| ------------------- | ------------------------------ |
| `for c in number:`  | 문자열 `number`의 각 문자를 하나씩 순회합니다. |
| `int(c)`            | 문자 숫자를 정수 숫자로 변환합니다.           |
| `answer += int(c)`  | 각 자리 숫자를 누적해서 더합니다.            |
| `return answer % 9` | 자리수 합을 9로 나눈 나머지를 반환합니다.       |

---

## Python 한 줄 풀이

```python
def solution(number):
    return sum(int(c) for c in number) % 9
```

## 예제 동작 과정

### 예제 1

```text
number = "123"
```

각 자리 숫자의 합은 다음과 같습니다.

```text
1 + 2 + 3 = 6
```

따라서 결과는 다음과 같습니다.

```text
6 % 9 = 6
```

결과:

```text
6
```

---

### 예제 2

```text
number = "78720646226947352489"
```

각 자리 숫자의 합은 다음과 같습니다.

```text
7 + 8 + 7 + 2 + 0 + 6 + 4 + 6 + 2 + 2 + 6 + 9 + 4 + 7 + 3 + 5 + 2 + 4 + 8 + 9 = 101
```

따라서 결과는 다음과 같습니다.

```text
101 % 9 = 2
```

결과:

```text
2
```

```text
시간 복잡도: O(N)
```

```text
공간 복잡도: O(1)
```

---

## 정리

이 문제는 큰 정수를 직접 숫자형으로 변환하지 않고, 문자열의 각 자리 숫자를 이용해 해결하는 문제입니다.

핵심은 다음과 같습니다.

```text
number % 9 == 각 자리 숫자의 합 % 9
```

따라서 문자열을 한 글자씩 순회하면서 각 자리 숫자를 더한 뒤, 마지막에 `9`로 나눈 나머지를 반환하면 됩니다.
