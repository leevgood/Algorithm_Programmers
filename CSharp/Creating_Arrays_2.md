# 0과 5로만 이루어진 정수 찾기

---

## 문제 설명

정수 `l`과 `r`이 주어졌을 때, `l` 이상 `r` 이하의 정수 중에서 숫자 **`0`**과 **`5`**로만 이루어진 모든 정수를 오름차순으로 저장한 배열을 반환하는 문제입니다.

만약 조건을 만족하는 정수가 없다면 `-1`이 담긴 배열을 반환합니다.

---

## 제한사항

| 조건  | 내용                  |
| --- | ------------------- |
| `l` | `1 ≤ l`             |
| `r` | `l ≤ r ≤ 1,000,000` |

---

## 입출력 예시

| l  | r   | result                            |
| -- | --- | --------------------------------- |
| 5  | 555 | `[5, 50, 55, 500, 505, 550, 555]` |
| 10 | 20  | `[-1]`                            |

---

## 문제 풀이 아이디어

이 문제는 `l`부터 `r`까지 모든 숫자를 확인하면서, 각 숫자가 **0과 5로만 이루어져 있는지** 검사하면 해결할 수 있습니다.

숫자를 문자열로 변환한 뒤, 각 문자가 `'0'` 또는 `'5'`인지 확인합니다.

예를 들어 `505`는 다음과 같이 검사됩니다.

```csharp
"505" → '5', '0', '5'
```

모든 문자가 `'0'` 또는 `'5'`이므로 조건을 만족합니다.

반면 `5501`은 `'1'`이 포함되어 있으므로 조건을 만족하지 않습니다.

---

## 소스 코드

```csharp
using System;
using System.Collections.Generic;
using System.Linq;

public class Solution
{
    public int[] solution(int l, int r)
    {
        // 조건을 만족하는 숫자를 저장할 리스트
        List<int> answer = new List<int>();

        // l부터 r까지 모든 숫자 검사
        for (int i = l; i <= r; i++)
        {
            // 현재 숫자가 0과 5로만 이루어져 있는지 확인
            if (IsOnlyZeroAndFive(i))
            {
                answer.Add(i);
            }
        }

        // 조건을 만족하는 숫자가 없다면 -1 반환
        if (answer.Count == 0)
        {
            return new int[] { -1 };
        }

        return answer.ToArray();
    }

    // 숫자가 0과 5로만 이루어져 있는지 검사하는 메서드
    private bool IsOnlyZeroAndFive(int number)
    {
        string str = number.ToString();

        foreach (char ch in str)
        {
            if (ch != '0' && ch != '5')
            {
                return false;
            }
        }

        return true;
    }
}
```

---

## 주요 메서드 및 문법 설명

| 문법 / 메서드       | 설명                             |
| -------------- | ------------------------------ |
| `List<int>`    | 크기가 동적으로 변하는 정수 리스트입니다.        |
| `Add()`        | 리스트에 값을 추가합니다.                 |
| `ToArray()`    | 리스트를 배열로 변환합니다.                |
| `ToString()`   | 숫자를 문자열로 변환합니다.                |
| `foreach`      | 문자열의 각 문자를 하나씩 순회할 때 사용합니다.    |
| `return false` | 조건을 만족하지 않을 경우 즉시 메서드를 종료합니다.  |
| `return true`  | 모든 검사를 통과했을 때 조건을 만족한다고 판단합니다. |

## 코드 동작 과정

예를 들어 `l = 5`, `r = 555`인 경우 다음과 같이 동작합니다.

```text
5   → 조건 만족
6   → 조건 불만족
...
50  → 조건 만족
55  → 조건 만족
...
500 → 조건 만족
505 → 조건 만족
550 → 조건 만족
555 → 조건 만족
```

따라서 결과는 다음과 같습니다.

```csharp
[5, 50, 55, 500, 505, 550, 555]
```

## 다른 풀이 방법

`0`과 `5`로만 이루어진 숫자는 개수가 많지 않습니다.

따라서 모든 숫자를 검사하지 않고, 가능한 숫자만 미리 생성하는 방식으로도 풀 수 있습니다.

예를 들어 다음과 같은 숫자들을 만들 수 있습니다.

```text
5
50
55
500
505
550
555
...
```

이 방식은 불필요한 숫자를 검사하지 않기 때문에 더 효율적입니다.

---

## 개선된 풀이 코드

```csharp
using System;
using System.Collections.Generic;

public class Solution
{
    public int[] solution(int l, int r)
    {
        List<int> answer = new List<int>();

        // 1부터 63까지 확인
        // 이진수로 변환했을 때 0과 1의 조합을 이용하여 0과 5로 이루어진 숫자를 생성
        for (int i = 1; i < 64; i++)
        {
            string binary = Convert.ToString(i, 2);

            // 이진수의 1은 5로, 0은 0으로 변환
            string numberString = binary.Replace('1', '5');

            int number = int.Parse(numberString);

            // 생성한 숫자가 l 이상 r 이하라면 추가
            if (number >= l && number <= r)
            {
                answer.Add(number);
            }
        }

        // 조건을 만족하는 숫자가 없다면 -1 반환
        if (answer.Count == 0)
        {
            return new int[] { -1 };
        }

        return answer.ToArray();
    }
}
```
### 파이썬 코드

```python
def solution(l, r):
    answer = []
    
    for num in range(l, r + 1):
        str_num = str(num)
        
        is_num = True
        
        for c in str_num:
            if c != '5' and c != '0':
                is_num = False
                break
        
        if is_num == True:
            answer.append(num)
        
    if len(answer) == 0:
        return [-1]
    
    return answer
```

---

## 개선된 풀이 설명

제한사항에서 `r`은 최대 `1,000,000`입니다.

따라서 가능한 숫자는 최대 6자리 정도까지만 확인하면 됩니다.

`1`부터 `63`까지의 숫자를 이진수로 변환하면 다음과 같은 패턴을 만들 수 있습니다.

| i | 이진수   | 변환 결과 |
| - | ----- | ----- |
| 1 | `1`   | `5`   |
| 2 | `10`  | `50`  |
| 3 | `11`  | `55`  |
| 4 | `100` | `500` |
| 5 | `101` | `505` |
| 6 | `110` | `550` |
| 7 | `111` | `555` |

즉, 이진수의 `1`을 `5`로 바꾸면 자연스럽게 **0과 5로만 이루어진 숫자**를 만들 수 있습니다.

---

## 시간 복잡도

### 기본 풀이

```text
O((r - l + 1) × 숫자의 자릿수)
```

`l`부터 `r`까지 모든 숫자를 검사하기 때문에 범위가 클수록 시간이 더 오래 걸립니다.

### 개선된 풀이

```text
O(1)
```

최대 63개의 숫자만 검사하므로 입력 범위와 관계없이 매우 빠르게 동작합니다.

---

## 참고 사항

* 숫자를 문자열로 바꾸면 각 자릿수를 쉽게 검사할 수 있습니다.
* `Convert.ToString(i, 2)`를 사용하면 정수를 이진수 문자열로 변환할 수 있습니다.
* 이진수 패턴을 활용하면 `0`과 `5`로 이루어진 숫자를 효율적으로 생성할 수 있습니다.
* 문제의 제한사항이 작을 때는 완전 탐색도 가능하지만, 가능한 숫자만 생성하는 방식이 더 깔끔하고 효율적입니다.
