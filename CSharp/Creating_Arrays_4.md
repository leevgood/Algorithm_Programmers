# 배열 만들기 4 - C# / Python 풀이

## 프로그램 설명

정수 배열 `arr`가 주어졌을 때, 조건에 따라 새로운 배열 `stk`를 만드는 문제입니다.

핵심 아이디어는 **스택(Stack)** 자료구조를 사용하는 것입니다.

`stk`의 마지막 원소가 현재 값 `arr[i]`보다 작으면 값을 추가하고,
크거나 같으면 마지막 원소를 제거합니다.

최종적으로 만들어진 `stk`를 반환합니다.

---

## 문제 핵심 조건

반복 중 다음 규칙을 따릅니다.

1. `stk`가 비어 있으면 `arr[i]`를 추가하고 `i`를 1 증가시킵니다.
2. `stk`의 마지막 원소가 `arr[i]`보다 작으면 `arr[i]`를 추가하고 `i`를 1 증가시킵니다.
3. `stk`의 마지막 원소가 `arr[i]`보다 크거나 같으면 `stk`의 마지막 원소를 제거합니다.
4. 이때 `i`는 증가하지 않습니다.

즉, 현재 값을 넣을 수 있을 때까지 `stk`의 뒤쪽 원소를 제거하는 방식입니다.

---

## C# 풀이

```csharp
using System;
using System.Collections.Generic;

public class Solution
{
    public int[] solution(int[] arr)
    {
        // 결과를 저장할 스택 역할의 리스트
        List<int> stk = new List<int>();

        // 배열 arr를 순회하기 위한 인덱스
        int i = 0;

        while (i < arr.Length)
        {
            // stk가 비어 있으면 현재 값을 추가
            if (stk.Count == 0)
            {
                stk.Add(arr[i]);
                i++;
            }
            // stk의 마지막 값이 arr[i]보다 작으면 추가
            else if (stk[stk.Count - 1] < arr[i])
            {
                stk.Add(arr[i]);
                i++;
            }
            // stk의 마지막 값이 arr[i]보다 크거나 같으면 제거
            else
            {
                stk.RemoveAt(stk.Count - 1);
            }
        }

        // List<int>를 int[] 배열로 변환하여 반환
        return stk.ToArray();
    }
}
```

## Python 풀이

```python
def solution(arr):
    # 결과를 저장할 스택
    stk = []

    # 배열 arr를 순회하기 위한 인덱스
    i = 0

    while i < len(arr):
        # stk가 비어 있으면 현재 값을 추가
        if not stk:
            stk.append(arr[i])
            i += 1

        # stk의 마지막 값이 arr[i]보다 작으면 추가
        elif stk[-1] < arr[i]:
            stk.append(arr[i])
            i += 1

        # stk의 마지막 값이 arr[i]보다 크거나 같으면 제거
        else:
            stk.pop()

    return stk
```

## 주요 메서드 및 문법 설명

| 언어     | 문법 / 메서드          | 설명                  |
| ------ | ----------------- | ------------------- |
| C#     | `List<int>`       | 크기가 동적으로 변하는 정수 리스트 |
| C#     | `Add()`           | 리스트 뒤에 값을 추가        |
| C#     | `RemoveAt(index)` | 특정 위치의 원소 제거        |
| C#     | `stk.Count - 1`   | 리스트의 마지막 인덱스        |
| C#     | `ToArray()`       | 리스트를 배열로 변환         |
| Python | `list`            | 여러 값을 저장하는 자료구조     |
| Python | `append()`        | 리스트 뒤에 값을 추가        |
| Python | `pop()`           | 리스트의 마지막 원소 제거      |
| Python | `stk[-1]`         | 리스트의 마지막 원소 접근      |

arr = [1, 4, 2, 5, 3]
```

## 출력 예시

[1, 2, 3]
```

## 코드 동작 과정

초기 상태:

```text
arr = [1, 4, 2, 5, 3]
stk = []
i = 0
```

동작 과정:

| i | arr[i] | stk 상태    | 동작                |
| - | -----: | --------- | ----------------- |
| 0 |      1 | []        | stk가 비어 있으므로 1 추가 |
| 1 |      4 | [1]       | 1 < 4 이므로 4 추가    |
| 2 |      2 | [1, 4]    | 4 >= 2 이므로 4 제거   |
| 2 |      2 | [1]       | 1 < 2 이므로 2 추가    |
| 3 |      5 | [1, 2]    | 2 < 5 이므로 5 추가    |
| 4 |      3 | [1, 2, 5] | 5 >= 3 이므로 5 제거   |
| 4 |      3 | [1, 2]    | 2 < 3 이므로 3 추가    |

최종 결과:

```text
[1, 2, 3]
```

## 시간 복잡도

```text
O(n)
```

각 원소는 `stk`에 최대 한 번 추가되고, 최대 한 번 제거됩니다.
따라서 전체 시간 복잡도는 `O(n)`입니다.

---

## 공간 복잡도

```text
O(n)
```

최악의 경우 모든 원소가 `stk`에 저장될 수 있으므로 공간 복잡도는 `O(n)`입니다.

---

## 참고 사항

이 문제는 단순 배열 처리처럼 보이지만, 실제로는 **스택(Stack)** 을 이용해 해결하는 문제입니다.

현재 값보다 크거나 같은 이전 값들을 제거하면서, 최종적으로 조건을 만족하는 증가 형태의 배열을 만들어 갑니다.
