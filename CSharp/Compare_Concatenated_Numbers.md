# 두 수 붙여서 비교하기 (Compare Concatenated Numbers)

---

## 프로그램 설명

이 문제는 양의 정수 `a`와 `b`가 주어졌을 때, 두 수를 문자열처럼 이어 붙인 값인 `a ⊕ b`와 `b ⊕ a` 중 더 큰 값을 반환하는 문제입니다.

예를 들어,

- `12 ⊕ 3 = 123`
- `3 ⊕ 12 = 312`

따라서 `123`과 `312`를 비교하면 `312`가 더 크므로 `312`를 반환합니다.

단, 두 값이 같다면 `a ⊕ b`를 반환합니다.

---

## 소스 코드

```csharp
using System;

public class Solution
{
    public int solution(int a, int b)
    {
        // 정수 a와 b를 문자열로 변환한 뒤 이어 붙입니다.
        string ab = a.ToString() + b.ToString();
        string ba = b.ToString() + a.ToString();

        // 이어 붙인 문자열을 정수로 변환합니다.
        int numAB = int.Parse(ab);
        int numBA = int.Parse(ba);

        // 두 값 중 더 큰 값을 반환합니다.
        // 값이 같으면 문제 조건에 따라 a ⊕ b 값을 반환합니다.
        if (numAB >= numBA)
        {
            return numAB;
        }

        return numBA;
    }
}
```

---

## 주요 메서드 및 문법 설명

| 문법 / 메서드 | 설명 |
|---|---|
| `ToString()` | 정수를 문자열로 변환합니다. |
| `+` | 문자열끼리 사용할 경우 두 문자열을 이어 붙입니다. |
| `int.Parse()` | 숫자로 이루어진 문자열을 정수로 변환합니다. |
| `if` 조건문 | 조건에 따라 다른 코드를 실행합니다. |
| `return` | 함수의 결과값을 반환합니다. |

---

## 입력 예시

```csharp
int a = 9;
int b = 91;
```

---

## 출력 예시

```csharp
991
```

---

## 코드 동작 과정

예시 입력:

```csharp
a = 9
b = 91
```

1. `a`와 `b`를 문자열로 변환합니다.

```csharp
a.ToString() = "9"
b.ToString() = "91"
```

2. 두 가지 방식으로 문자열을 이어 붙입니다.

```csharp
ab = "9" + "91"  // "991"
ba = "91" + "9"  // "919"
```

3. 문자열을 정수로 변환합니다.

```csharp
numAB = 991
numBA = 919
```

4. 두 값을 비교합니다.

```csharp
991 >= 919
```

5. 더 큰 값인 `991`을 반환합니다.

## 참고 사항

- 제한사항에서 `1 ≤ a, b < 10,000`이므로, 두 수를 이어 붙여도 최대 길이는 8자리입니다.
- 따라서 `int` 자료형으로 충분히 처리할 수 있습니다.
- 문자열로 변환하여 이어 붙인 뒤 숫자로 비교하면 문제를 간단하게 해결할 수 있습니다.
