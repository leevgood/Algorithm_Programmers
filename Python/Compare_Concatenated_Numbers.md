# 피타고라스의 정리로 다른 한 변의 제곱 구하기 (Pythagorean Theorem)

---

## 프로그램 설명

이 문제는 직각삼각형에서 한 변의 길이 `a`와 빗변의 길이 `c`가 주어졌을 때, 다른 한 변의 길이의 제곱인 `b_square`를 구하는 문제입니다.

직각삼각형에서는 피타고라스의 정리에 따라 다음 공식이 성립합니다.

```text
a² + b² = c²
```

따라서 다른 한 변의 제곱 `b_square`는 다음과 같이 구할 수 있습니다.

```text
b² = c² - a²
```

---

## 소스 코드

```python
# 한 변의 길이 a를 입력받습니다.
a = int(input())

# 빗변의 길이 c를 입력받습니다.
c = int(input())

# 피타고라스의 정리를 이용하여 다른 한 변의 제곱을 구합니다.
b_square = c ** 2 - a ** 2

# 결과를 출력합니다.
print(b_square)
```

---

## 주요 메서드 및 문법 설명

| 문법 / 메서드 | 설명 |
|---|---|
| `input()` | 사용자로부터 문자열 입력을 받습니다. |
| `int()` | 입력받은 문자열을 정수로 변환합니다. |
| `**` | 거듭제곱 연산자입니다. 예를 들어 `c ** 2`는 `c`의 제곱을 의미합니다. |
| `print()` | 결과값을 화면에 출력합니다. |

---

## 입력 예시 1

```text
3
5
```

---

## 출력 예시 1

```text
16
```

---

## 코드 동작 과정 1

입력값은 다음과 같습니다.

```python
a = 3
c = 5
```

피타고라스의 정리를 사용합니다.

```text
a² + b² = c²
```

다른 한 변의 제곱 `b²`를 구하기 위해 식을 변형합니다.

```text
b² = c² - a²
```

값을 대입하면 다음과 같습니다.

```text
b² = 5² - 3²
b² = 25 - 9
b² = 16
```

따라서 출력 결과는 `16`입니다.

---

## 입력 예시 2

```text
9
10
```

---

## 출력 예시 2

```text
19
```

---

## 코드 동작 과정 2

입력값은 다음과 같습니다.

```python
a = 9
c = 10
```

계산 과정은 다음과 같습니다.

```text
b² = c² - a²
b² = 10² - 9²
b² = 100 - 81
b² = 19
```

따라서 출력 결과는 `19`입니다.

---

## 참고 사항

- 제한사항은 `1 ≤ a < c ≤ 100`입니다.
- 빗변 `c`가 항상 한 변 `a`보다 크기 때문에 `c² - a²`는 항상 양수입니다.
- 이 문제에서는 다른 한 변의 길이 `b`가 아니라, 다른 한 변의 길이의 제곱인 `b_square`를 출력해야 합니다.
- 따라서 제곱근을 구할 필요 없이 `c ** 2 - a ** 2`만 계산하면 됩니다.
