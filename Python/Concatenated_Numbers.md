# 홀짝 이어 붙이기 - Python 풀이

## 문제 설명

정수가 담긴 리스트 `num_list`가 주어집니다.

`num_list`에서 **홀수만 순서대로 이어 붙인 수**와 **짝수만 순서대로 이어 붙인 수**를 만든 뒤, 두 수의 합을 반환하는 문제입니다.

예를 들어,

```python
num_list = [3, 4, 5, 2, 1]
```

홀수만 이어 붙이면 다음과 같습니다.

```text
3, 5, 1 → 351
```

짝수만 이어 붙이면 다음과 같습니다.

```text
4, 2 → 42
```

따라서 결과는 다음과 같습니다.

```text
351 + 42 = 393
```

---

## 제한사항

- `2 ≤ num_list의 길이 ≤ 10`
- `1 ≤ num_list의 원소 ≤ 9`
- `num_list`에는 적어도 한 개씩의 짝수와 홀수가 있습니다.

---

## 입출력 예시

| num_list | result |
|---|---:|
| `[3, 4, 5, 2, 1]` | `393` |
| `[5, 7, 8, 3]` | `581` |

---

## 소스 코드

```python
def solution(num_list):
    even = ""
    odd = ""
    
    for num in num_list:
        # 짝수인 경우 even 문자열에 추가
        if num % 2 == 0:
            even += str(num)
        # 홀수인 경우 odd 문자열에 추가
        else:
            odd += str(num)
    
    # 문자열로 이어 붙인 홀수와 짝수를 정수로 변환한 뒤 더함
    return int(odd) + int(even)
```

---

## 코드 설명

### 1. 홀수와 짝수를 저장할 문자열 생성

```python
even = ""
odd = ""
```

짝수만 이어 붙일 문자열 `even`과 홀수만 이어 붙일 문자열 `odd`를 빈 문자열로 초기화합니다.

---

### 2. 리스트의 숫자를 하나씩 확인

```python
for num in num_list:
```

`num_list`에 들어 있는 숫자를 처음부터 끝까지 하나씩 확인합니다.

---

### 3. 짝수와 홀수 구분

```python
if num % 2 == 0:
    even += str(num)
else:
    odd += str(num)
```

`num % 2 == 0`이면 짝수입니다.

짝수라면 `even` 문자열에 추가하고, 홀수라면 `odd` 문자열에 추가합니다.

이때 숫자를 문자열에 이어 붙여야 하므로 `str(num)`을 사용합니다.

---

## 예시 동작 과정

입력값이 다음과 같다고 가정합니다.

```python
num_list = [3, 4, 5, 2, 1]
```

반복문이 실행되면서 값은 다음처럼 변합니다.

| num | 홀수/짝수 | odd | even |
|---:|---|---|---|
| `3` | 홀수 | `"3"` | `""` |
| `4` | 짝수 | `"3"` | `"4"` |
| `5` | 홀수 | `"35"` | `"4"` |
| `2` | 짝수 | `"35"` | `"42"` |
| `1` | 홀수 | `"351"` | `"42"` |

최종적으로 다음과 같이 계산됩니다.

```python
int(odd) + int(even)
```

```text
int("351") + int("42")
= 351 + 42
= 393
```

---

## 형 변환 주의점

Python에서는 C#처럼 아래와 같이 형 변환하지 않습니다.

```python
(str)num
(int)even
```

위 코드는 Python 문법이 아니므로 오류가 발생합니다.

Python에서는 다음과 같이 함수 형태로 형 변환해야 합니다.

```python
str(num)
int(even)
int(odd)
```

## 다른 풀이 방법

리스트 컴프리헨션과 `join()`을 사용하면 더 짧게 작성할 수 있습니다.

```python
def solution(num_list):
    odd = ''.join(str(num) for num in num_list if num % 2 == 1)
    even = ''.join(str(num) for num in num_list if num % 2 == 0)
    
    return int(odd) + int(even)
```

---

## 주요 문법 정리

| 문법 | 설명 |
|---|---|
| `for num in num_list` | 리스트의 원소를 하나씩 꺼내는 반복문 |
| `%` | 나머지를 구하는 연산자 |
| `num % 2 == 0` | 숫자가 짝수인지 확인하는 조건 |
| `str(num)` | 숫자를 문자열로 변환 |
| `int(문자열)` | 문자열을 정수로 변환 |
| `+=` | 기존 값에 새로운 값을 더해서 저장 |

---

## 풀이 핵심

이 문제의 핵심은 다음과 같습니다.

1. 리스트를 순서대로 순회한다.
2. 홀수와 짝수를 구분한다.
3. 각각 문자열로 이어 붙인다.
4. 마지막에 정수로 변환한 뒤 더한다.

문제에서 숫자의 순서를 유지해야 하므로, 정렬하거나 순서를 바꾸면 안 됩니다.

---

## 시간 복잡도

```text
O(n)
```

리스트 `num_list`를 한 번 순회하므로 시간 복잡도는 `O(n)`입니다.

## 공간 복잡도

```text
O(n)
```

홀수와 짝수를 문자열로 저장하므로 공간 복잡도는 `O(n)`입니다.
